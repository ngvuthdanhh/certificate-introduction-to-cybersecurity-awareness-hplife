# Cybersecurity Awareness Glossary

| Term | Definition |
|------|-------------|
| **Authentication** | Process of verifying the identity of a user or system. |
| **Backup** | Copy of data stored to prevent loss in case of system failure. |
| **Data Breach** | Unauthorized access and retrieval of sensitive information. |
| **Encryption** | Conversion of data into a coded format to prevent unauthorized access. |
| **Firewall** | Network security system that monitors and controls incoming and outgoing traffic. |
| **Malware** | Software designed to damage, disrupt, or gain unauthorized access to systems. |
| **Phishing** | Fraudulent attempt to obtain sensitive information by disguising as a trustworthy entity. |
| **Ransomware** | Type of malware that locks data and demands payment to restore access. |
| **Social Engineering** | Psychological manipulation used to trick individuals into revealing confidential information. |
| **Two-Factor Authentication (2FA)** | Security process requiring two verification methods to access an account. |
| **VPN (Virtual Private Network)** | Technology that creates a secure and encrypted connection over the internet. |
| **Zero Trust** | Security model assuming no implicit trust; verifies every access request. |
